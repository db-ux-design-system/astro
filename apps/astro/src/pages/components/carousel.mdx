# Carousel Component

The `Carousel` component renders a carousel of images with optional captions.

If using Astro in a static or SSR environment, you must enable client-side rendering. Use **client:load** to hydrate the component on the client. See Astro's [documentation](https://docs.astro.build/en/reference/directives-reference/#client-directives) for details:

```tsx
import { Carousel } from "@db-ux/astro/components";

<Carousel
  client:only="react"
  items={[
    { src: "example/cat.jpg", caption: "I am a cat." },
    { src: "example/cat-dog.jpg", caption: "We are a dog and a cat." },
    { src: "example/chicken.jpg" },
  ]}
/>;
```
