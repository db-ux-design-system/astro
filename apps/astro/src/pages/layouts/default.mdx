## Introduction

The default layout is a versatile and responsive layout designed to cover most common use cases (like documentation). It includes a header, footer and a main content area. It will also render a table of contents if your page contains headings. Most pages on this site use the default layout (including this one).

## How to use

Markdown (`.md`) or MDX (`.mdx`):

```yaml
---
layout: '@db-ux/astro/layouts/default'
---
```

Astro components (`.astro`):

```astro
---
import DefaultLayout from '@db-ux/astro/layouts/default';
---
<DefaultLayout>
  <!-- Your page content goes here -->
</DefaultLayout>
```

## Front Matter attributes

```yaml
---
title: string          # The title of the page, displayed in the browser, as the first level heading and used for SEO.
description: string    # A brief description of the page, used for SEO.
toc: boolean           # Whether to display a table of contents for the page. Default is true.
---
```
