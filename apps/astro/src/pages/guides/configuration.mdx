## Astro Configuration

Astro allows you to customize various aspects (including integrations) of your project through the `astro.config.mjs` file located at the root of your project. Instead of going through every possible configuration option, this guide focuses on the key settings relevant to using `@db-ux/astro`. If you want to learn more about Astro's configuration options in general, please refer to the [official Astro documentation](https://docs.astro.build/en/reference/configuration-reference/).
